Debugger - Just My Code

  • Step only in your own code
  • Break in caught exceptions in your code only
  • Break next in your code
  • Mark your files:
    • File picker, tab header, MyCode.json

Debugger - Source Maps

  • F12 now supports the Source Maps v3 spec!
    • Syntax highlighting for TypeScript, CoffeeScript and Script #
    • Dynamically enable and disable
  • Support for //# sourceURL to label eval code

Never forget!

  • The debugger keeps track of your state so when you close F12 it's the same when you bring it back up
  • Watches, breakpoints, open files, pretty printed files, source map and JMC settings

Shortcut Keys - Pro Tip

  • When broken in the debugger use ctrl + shift + F5 to abort and refresh the page
  • Move between tools with ctrl + [ or ctrl + ]
  • Don't forget ctrl + # to jump to a tool (e.g. DOM Explorer is ctrl + 1)

Console - Logging

  • console.log logs objects
  • console.log includes printf style string formatting
  • Errors can be logged before F12 is open

Console - Execution

  • Easy way to pick iframes to execute against
  • New console cmd $_ for last expression
  • Auto complete now includes locals

Memory

  • New workflow for scenarios where you need to identify "left over" objects
    • New scope filter
  • Other improvements
    • Function source nav
    • Gridlines
    • Additional settings options
    • Show in dominators

Profiling - UI

(Will demo in Phone Demo)
  • New filters for background and network activity
  • Scope to events

DOM Explorer - Changebars

  • Keep track of all the CSS changes you make through the tools
  • New "Changes" tab show all the changes
  • Revert the item back to it's original state

DOM Explorer - Pseudo States

  • Support for forcing :hover and :visited
  • Like all of the DOM Explorer the feature is also in VS for Store Apps

Windows Phone 8.1 Phone Debugging



Full stack of Visual Studio tools for HTML/JS are now enabled for IE11 on Windows Phone 8.1 in Visual Studio 2012 Update 2

Summary


  • New F12 features in the Windows update for Win7 and Win8.1 in the 4/8 Windows Update
  • Debugging IE on Windows Phone 8.1 (device & emulator)
